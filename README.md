# Technical Assignment — Full Stack

## Objective

This task is designed to evaluate your **problem understanding, code structure, Git usage, and communication**.
Perfection is not expected. Clarity and effort matter more.

---

## Task Overview

Build a **simple CRUD application** for managing a list of items.

You may choose **any one stack**:

* React/next.js/angular (frontend) + mock data / local state
  **OR**
* Node.js / Express (API only)

Choose what you are most comfortable with.

---

## Functional Requirements

Your application should support:

1. View a list of items
2. Add a new item
3. Edit an existing item
4. Delete an item

**Item fields (minimum):**

* Title
* Description
* Image/video (media)
* Created Date (auto-generated)
